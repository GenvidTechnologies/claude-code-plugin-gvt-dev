# raw/

This directory holds **immutable captured sources** for the project wiki
maintained by `/gvt-dev:maintain-wiki` — transcripts, doc snapshots, decision
records, data dumps, or anything else worth ingesting into `wiki/`, exactly as
captured.

**Never edit a file in this directory.** If a source has changed, re-capture
it as a new file rather than modifying the existing one — the old capture
stays as the historical record of what was true when it was captured.

See `docs/wiki-schema.md` for the full maintenance schema (page format,
ingest/query/lint verbs, and the decay policy).
